# Кейс 5

В рамках задания реализована программа на `Python 3`, которая генерирует HTML-страницу с информацией об организации на базе практики. Скрипт `generate_page.py` формирует файл `index.html`, содержащий название организации, основные сведения о ней, CSS-оформление, основной шрифт и 5 дополнительных вариантов шрифтов.

Решение соответствует условию кейса, так как позволяет автоматически получить готовую веб-страницу без ручной верстки итогового файла. Для запуска необходимо выполнить команду:

```bash
python3 generate_page.py
```
