# Кейс 4

В рамках задания реализован набор unit-тестов на `GoogleTest` для классов `Queue`, `Heap` и `BinaryTree` с требуемыми операциями `push`, `pop` и `search`. Проект написан на `C++17`, сборка выполняется через `CMake`, что позволяет автоматически собрать исполняемый файл с тестами и проверить корректность работы структур данных.

Тесты покрывают базовые и граничные сценарии: работу с пустыми структурами, добавление и удаление элементов, корректность порядка обработки данных, поиск в дереве и обработку ошибочных ситуаций.

## Сборка и запуск

```bash
mkdir build
cd build
cmake ..
cmake --build .
./practice_tests
```
